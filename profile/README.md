## What is Context Foundry?

Context Foundry is an **MCP (Model Context Protocol) server** that empowers Claude Code CLI to build complete software projects autonomously.

**Real Example:**
```
User: "Build a Mario platformer game in JavaScript"
[User walks away for 7 minutes]
Result: ✅ Complete game deployed to GitHub, all tests passing
```

## Get Started

**👉 [Visit the main repository for complete documentation →](https://github.com/context-foundry/context-foundry)**

**Quick Start:**
1. Install: `pip install -r requirements-mcp.txt`
2. Configure Claude Code MCP
3. Start building: Just describe what you want!
